Intel Celeron G5905 3.50GHz

Los nuevos procesadores Intel® Core™ de 10ª generación ofrecen mejoras de rendimiento notables para conseguir una productividad mejorada y un entretenimiento impresionante, incluyendo Intel® Wi-Fi 6 (Gig+), tecnología Thunderbolt™ 3, HDR 4K, optimización de sistema inteligente y mucho más.

Características:

  • Admite procesadores Intel® Core ™ de 10.a generación, Intel® Core ™ de 11.a generación, Pentium® Gold y Celeron® para zócalo LGA 1200
  • dmite memoria DDR4, hasta 5066 (OC) MHz
  • Core Boost: con diseño premium y diseño de energía digital para admitir más núcleos y proporcionar un mejor rendimiento.
  • DDR4 Boost: tecnología avanzada para entregar señales de datos puros para el mejor rendimiento y estabilidad.
  • Experiencia ultrarrápida: PCIe 4.0, Lightning Gen4 x4 M.2 con M.2 Shield Frozr
  • LAN 2.5G: Solución de red mejorada para uso profesional y multimedia. Ofrece una conexión de red segura, estable y rápida
  • Aumento de audio: recompense sus oídos con calidad de sonido de estudio
  • Steel Armor: protege las tarjetas VGA contra la flexión y las interferencias electromagnéticas para un mejor rendimiento, estabilidad y resistencia.
